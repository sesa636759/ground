# set-app-avatar

## Properties

| Property | Type | Description |
| --- | --- | --- |
| **alt** | `string` | Alternative text for the image |
| **gap** | `number` | Letter type unit distance between left and right sides |
| **icon** | `string` | Custom icon for an icon avatar |
| **shape** | `'circle' \| 'square'` | The shape of avatar |
| **size** | `'xs' \| 'small' \| 's' \| 'default' \| 'm' \| 'large' \| 'l' \| 'xl' \| number` | The size of the avatar |
| **src** | `string` | The address of the image for an image avatar |
| **srcSet** | `string` | A list of sources to use for different screen resolutions |
| **imgDraggable** | `boolean \| 'true' \| 'false'` | Whether the picture is allowed to be dragged |
| **crossOrigin** | `'anonymous' \| 'use-credentials' \| ''` | CORS settings attributes |
| **bgColor** | `string` | Background color for text/icon avatar |
| **color** | `string` | Text color for text/icon avatar |
| **border** | `string` | Border for avatar |
| **tooltipText** | `string` | Tooltip text to display on hover |
| **xs** | `number` |  |
| **sm** | `number` |  |
| **md** | `number` |  |
| **lg** | `number` |  |
| **xl** | `number` |  |
