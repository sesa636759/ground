# Component Documentation

This directory contains automatically generated documentation for all UI components in the project.
The documentation is generated from the type definitions in `node_modules` and includes:

- **Properties**: Name, Type, Description
- **Events**: Name, Detail Type, Description
- **Usage Examples**: Basic HTML usage

## Updating Documentation

To update these files (e.g., after upgrading the UI library), run the `generate-docs.js` script in the root directory:

```bash
node generate-docs.js
```
