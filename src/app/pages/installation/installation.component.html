Get Started with Ground UI

Install the most powerful and beautiful Angular UI component library. Build stunning applications faster with our highly customizable design system.

1

Prerequisites

Ensure you have Node.js and Angular CLI installed

Node.js 18.x + Angular 17+ NPM or Yarn
2

Install Package

Add the library to your Angular project

bash
$ npm install ng-ground-ui
$ yarn add ng-ground-ui
3

Import Styles

Add the global theming SCSS to your angular.json or main styles.scss

scss
// In your global styles.scss @import 'ng-ground-ui/styles/themes/light.scss'; @import 'ng-ground-ui/styles/core.scss';
4

Usage

Start using components right away

typescript
import { Component } from '@angular/core'; import { PgButtonModule } from 'ng-ground-ui'; @Component({ selector: 'app-root', standalone: true, imports: [PgButtonModule], template: `<pg-button variant="primary">Click Me</pg-button>` }) export class AppComponent {}
