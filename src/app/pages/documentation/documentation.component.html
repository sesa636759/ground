Web Components

Vanilla JS / HTML usage with no build step required. Zero dependencies.

Supported

Theming

GROUND UI is fully themeable via CSS custom properties. Override any token to match your brand.

Design Tokens

Every color, size and shadow is a CSS variable you can override globally or per-component.

Dark Mode

Built-in dark theme activated with a single data-theme="dark" attribute on your root element.

Custom Palettes

Replace the primary palette entirely. Every component adapts automatically.
