Panel

A content container that supports themes, dragging, resizing, modals, and window-like behaviors.

{{ activeExample | titlecase }} Features

Simple container with header, content, and footer slots. Built-in color variants for different contexts. Display count or status indicators in the header. Users can reposition the panel by dragging the header. Resize handles allow users to customize dimensions. Window-like controls for minimizing and maximizing content. Frosted glass effect using backdrop-filter blur. Overlay panel that demands user attention with a backdrop.

Panels provide a consistent container for organising related content. They support rich HTML content via slots.
