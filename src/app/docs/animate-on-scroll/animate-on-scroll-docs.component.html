Properties API

Property Type Default Description
animation string 'fade-in' Animation type (fade-in, slide-up, zoom-in, etc.)
duration string '600ms' Animation duration
delay string '0ms' Animation delay before starting
threshold number 0.1 Percentage of element visibility before animation triggers
easing string 'ease' CSS easing function
once boolean true Whether animation should happen only once
offset number 0 Offset in pixels from trigger point
disabled boolean false Disable animations
