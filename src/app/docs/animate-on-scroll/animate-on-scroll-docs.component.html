Directive

Apply the appAnimateOnScroll directive to any element to add scroll-triggered animations.

Input Properties

Property Type Default Description
animation AnimationType 'fade-in' Type of animation to apply
delay number 0 Delay before animation starts (in milliseconds)
duration number 600 Animation duration (in milliseconds)
threshold number 0.1 Intersection threshold (0-1) - how much of element must be visible
once boolean true If true, animates only once. If false, animates every time element enters viewport

Animation Types

Type Description
'fade-in' Element fades in
'slide-up' Element slides up from below
'slide-down' Element slides down from above
'slide-left' Element slides from right to left
'slide-right' Element slides from left to right
'zoom-in' Element scales up from smaller size
'zoom-out' Element scales down from larger size
'flip-up' Element flips up with 3D perspective
'flip-down' Element flips down with 3D perspective
'rotate-in' Element rotates and scales in

Usage Examples

// Basic usage
<div appAnimateOnScroll>Content</div>

// With animation type
<div appAnimateOnScroll animation="slide-up">Content</div>

// With delay
<div appAnimateOnScroll animation="fade-in" [delay]="300">Content</div>

// Custom duration and threshold
<div appAnimateOnScroll 
     animation="zoom-in" 
     [duration]="800" 
     [threshold]="0.5">Content</div>

// Repeat animation
<div appAnimateOnScroll animation="fade-in" [once]="false">Content</div>

// Staggered animations
<div *ngFor="let item of items; let i = index"
     appAnimateOnScroll
     animation="slide-up"
     [delay]="i * 100">
  {{ item }}
</div>
