📖 Documentation

Overview

The ui-accordion component provides a powerful, feature-rich accordion with 15+ advanced capabilities including search, drag & drop, nested items, persistence, and more.

Features

  • ✅ Subtitle Support - Add secondary text below titles
  • 🔍 Search & Filter - Real-time search with highlighting
  • ⚡ Action Buttons - Add interactive buttons to headers
  • 🌳 Nested Accordions - Hierarchical accordion structures
  • 🎯 Drag & Drop - Reorder items with drag and drop
  • 💾 Persistence - Save state to localStorage
  • ⏳ Loading States - Show loading indicators
  • 🌍 RTL Support - Full right-to-left language support
  • 📦 Dense Mode - Compact layout for data-heavy UIs
  • 📡 Animation Events - beforeOpen, afterOpen, beforeClose, afterClose
  • 🎨 Multiple Variants - default, bordered, splitted, light, shadow
  • 📏 Size Options - sm, md, lg
  • 🔒 Disabled States - Disable entire accordion or individual items
  • ♿ Accessibility - Full keyboard navigation and screen reader support

Basic Usage

<ui-accordion id="myAccordion"></ui-accordion>

<script>
  const accordion = document.getElementById('myAccordion');
  accordion.items = JSON.stringify([
    {{ '{' }}
      id: 'item1',
      title: 'First Item',
      subtitle: 'Optional subtitle',
      content: 'Content goes here',
      icon: '📄'
    {{ '}' }}
  ]);
</script>

Events

// Listen to change events
accordion.addEventListener('accordionChange', (e) => {{ '{' }}
  console.log('Item toggled:', e.detail.itemId);
  console.log('Is open:', e.detail.isOpen);
{{ '}' }});

// Listen to animation lifecycle events
accordion.addEventListener('beforeOpen', (e) => {{ '{' }}
  console.log('About to open:', e.detail.itemId);
{{ '}' }});

accordion.addEventListener('afterOpen', (e) => {{ '{' }}
  console.log('Finished opening:', e.detail.itemId);
{{ '}' }});

// Listen to action button clicks
accordion.addEventListener('accordionAction', (e) => {{ '{' }}
  console.log('Action clicked:', e.detail.actionId);
  console.log('On item:', e.detail.itemId);
{{ '}' }});

Accessibility

The accordion component is fully accessible:

  • Full keyboard navigation (Arrow keys, Home, End, Enter, Space)
  • ARIA attributes for screen readers
  • Focus management
  • Reduced motion support
  • Print-friendly styles
