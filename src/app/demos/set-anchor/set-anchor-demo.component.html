01. Introduction

Click any anchor item above and notice the URL hash changes (e.g., `#hash-intro`). You can share URLs with specific sections, and they'll scroll automatically when the page loads.

02. Features

The `update-hash` property enables this behavior. Key benefits: Browser back/forward buttons work, Deep linking to specific sections, Shareable URLs with section context.

03. Examples

Perfect for: Documentation sites, Long-form content and articles, Educational materials, Landing pages with multiple sections.

04. API Reference

Enable URL synchronization with the `update-hash` attribute. Each section gets a unique, bookmarkable URL.

1. Setup
