🚀 Advanced

Hash synchronization, keyboard navigation, and nested items

Hash/URL Synchronization

01. Introduction

Click any anchor item above and notice the URL hash changes (e.g., `#hash-intro`). You can share URLs with specific sections, and they'll scroll automatically when the page loads.

02. Features

The `update-hash` property enables this behavior. Key benefits: Browser back/forward buttons work, Deep linking to specific sections, Shareable URLs with section context.

03. Examples

Perfect for: Documentation sites, Long-form content and articles, Educational materials, Landing pages with multiple sections.

04. API Reference

Enable URL synchronization with the `update-hash` attribute. Each section gets a unique, bookmarkable URL.

Keyboard Navigation

1. Setup

Try it: Click on any anchor item above, then use your arrow keys ↑↓ to navigate. The keyboard navigation improves accessibility and allows power users to navigate quickly without using the mouse.

2. Configure

Arrow navigation provides better accessibility compliance and keyboard-only navigation support. WCAG 2.1 compliant, Screen reader friendly, No mouse required.

3. Build

Navigate efficiently without reaching for your mouse - perfect for developers and power users.

4. Deploy

Press Enter on the focused item to scroll smoothly to the section. The anchor component handles all keyboard events and focus management automatically.
