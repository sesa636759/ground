Setup

When testing components like Label or app-label, you'll need a stable testing environment. We recommend using @testing-library/angular because it encourages testing from a user's perspective.

Configuration

You may also need to configure global providers if your Labels depend on specific theme providers or localization injection tokens.

Optional: For simple components like Label, global providers might be optional unless you rely on custom theme injection.
