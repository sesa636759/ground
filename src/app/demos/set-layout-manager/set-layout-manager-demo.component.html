🔧 API Methods

Programmatic control over layouts

addPanel()

Add a new panel to a region

await layout.addPanel({{ '{' }} regionId, title, html {{ '}' }})

removePanel()

Remove a panel by ID

await layout.removePanel(panelId)

movePanel()

Move panel to another region

await layout.movePanel(panelId, targetRegionId)

splitRegion()

Split a region into two

await layout.splitRegion(regionId, 'horizontal')

popoutPanel()

Detach panel to floating window

await layout.popoutPanel(panelId)

maximizePanel()

Maximize panel to full layout

await layout.maximizePanel(panelId)

addRegion()

Add a new region to layout

await layout.addRegion({{ '{' }} id, title, size {{ '}' }})

getLayoutState()

Get current layout configuration

const state = await layout.getLayoutState()
