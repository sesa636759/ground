💡 Recommended Use Cases

Choose app-ui-library When:

  • Building complex data visualization dashboards
  • Need advanced table features (sorting, filtering, export)
  • Creating form-heavy applications
  • Require maximum customization options
  • Building analytics or business intelligence platforms

Choose set-ui-lib When:

  • Building administrative interfaces & panels
  • Need flexible layout management systems
  • Creating drag-and-drop dashboards
  • Want modern, polished UI components
  • Building responsive admin panels with custom layouts

🎯 Best Practices & Tips

For Performance

  • Use virtual scrolling for large datasets in tables
  • Implement lazy loading where applicable
  • Optimize chart rendering with proper data chunking
  • Use skeleton loaders during data fetches
  • Leverage memoization for heavy components

For Development

  • Utilize variant system for consistent theming
  • Implement proper event handling for interactions
  • Use public methods for programmatic control
  • Leverage built-in validation on form inputs
  • Use icon libraries consistently across components
