# set-app-card

## Properties

| Property | Type | Description |
| --- | --- | --- |
| **cardTitle** | `string` | Card title |
| **extra** | `string` | Content to show in the extra section of the header |
| **size** | `'default' \| 'small'` | Card size |
| **bordered** | `boolean` | Whether to show border |
| **hoverable** | `boolean` | Hoverable effect with shadow |
| **loading** | `boolean` | Shows a loading skeleton |
| **type** | `'inner' \| 'default'` | Card type |
| **layout** | `'vertical' \| 'horizontal'` | Card layout orientation |
| **glass** | `boolean` | Enable glassmorphism effect (premium visual) |

## Events

No events found.

## Usage Example

```html
<set-app-card
  cardTitle="value"
  extra="value"
  [size]="value"
></set-app-card>
```
